## ⚡ FASTEST WAY TO START (5 Minutes)

### 1️⃣ Start XAMPP (1 minute)
- Open: `C:\xampp\xampp-control.exe`
- Click **Start** next to Apache
- Click **Start** next to MySQL
- Wait for **GREEN** status ✅

### 2️⃣ Import Database (2 minutes)
- Open: http://localhost/phpmyadmin
- Click **New**
- Name: `dumbledore_boutique`
- Click **Create**
- Go to **Import** tab
- Select: `database/database.sql`
- Click **Import** ✅

### 3️⃣ Open Your Website (1 minute)
```

👨‍💼 ADMIN:     http://localhost/dumbeldoreBOUTIQUE/admin/
```

## 🆘 QUICK TROUBLESHOOTING

### ❌ "Connection Refused"
→ Make sure MySQL is running (green in XAMPP)

### ❌ "Database not found"
→ Import database.sql (follow step 2 above)

### ❌ Login not working
→ Use admin@dumbledore.com / admin@123

### ❌ Page shows error
→ Clear browser cache (Ctrl+Shift+Delete)

### ❌ XAMPP won't start Apache
→ Check port 80 is free, or change port in XAMPP
